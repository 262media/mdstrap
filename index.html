About

jQuery mdStrap plugin creates a side menu for smartphones or other handheld devices. It's built based on Twitter Bootstrap's framework version 4 (currently alpha 2).

Plugin installation

You'll have to include the CSS in the <head> and JavaScript just before your </body> of your project and that's it.

                            
  <!-- Bootstrap CSS -->
<link rel="stylesheet" href="https://cdn.rawgit.com/twbs/bootstrap/v4-dev/dist/css/bootstrap.css" />

  <!-- mdStrap Menu -->
<link rel="stylesheet" href="css/mdstrap.css" />

Options

Name Type/Default Description
fixedTop Bool (true by default) Setting it to true allowes you to customise the appearence of the menu when scrolling.
menuTrigger CSS class (navbar-toggler by default) The class for the button that will trigger the menu when on mobile devices.
scrollValue integer (10 by default) If fixedTop is set to true, this is required to trigger the necessary class for stylize the navbar.
mobileMenuBreak integer (992 by default) In px, define the width breakpoint on which the mobile version of the menu kicks in.
